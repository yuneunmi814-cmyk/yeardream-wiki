## 1. 입력 – input()

### 입력

print를 이용해서 컴퓨터로부터 정보를 받았다면, 정보를 컴퓨터에게 전달할 수 있음

> (도식) 사용자 → `input` → 컴퓨터, 컴퓨터 → `print` → 사용자

컴퓨터는 사용자가 전달한 값(자료)을 어딘가에 보관해야 함 → 변수를 사용!

### input() 기본 사용법

`변수 = input()`. 터미널에 값을 "입력" + Enter

```python
var = input()
# 실행 결과
# /* 코드가 실행되는 중입니다! */
# > 터미널 입력 값을 넣어주세요.
```

### input()의 중요한 특징 (1) — 문자열로 입력

무엇을 입력하든 "문자열"로 입력 받아짐

```python
var = input("숫자를 입력하세요 : ")
print(var)
print(type(var))
# 실행 결과
# 숫자를 입력하세요 : 10
# 10
# <class 'str'>
```

### input()의 중요한 특징 (2) — 가공 필요

입력 받은 자료를 자유롭게 사용하기 위해서는 가공이 필요

```python
var1 = input("숫자를 입력하세요 : ")
var2 = input("숫자를 입력하세요 : ")
print(var1 + var2)
# 실행 결과
# 숫자를 입력하세요 : 100
# 숫자를 입력하세요 : 300
# 100300
```

### 어떻게 가공해 주지? — 형 변환

만약, 문자열을 숫자로 바꾸고 싶다면? → 자료형을 변환(형 변환)을 사용

```python
var1 = "8"
var2 = "1000"
print(var1 + var2)
print(int(var1) + int(var2))
# 실행 결과
# 81000
# 1008
```

### 형 변환

| 구분 | 이름 | 예시 | 결과 |
|---|---|---|---|
| 정수 | integer | `int("1008")` | `1008` |
| 실수 | float | `float("3.14159")` | `3.14159` |
| 문자열 | string | `str(3.14159)` | `"3.14159"` |
| 리스트 | list | `list("1008")` | `["1", "0", "0", "8"]` |

```python
a = "345"
b = int("345")

print(a, b)  # 345 345

print(type(a))  # <class 'str'> | 문자열
print(type(b))  # <class 'int'> | 숫자
```

## 2. 논리형 자료와 비교연산

### 논리 자료형

참(True) 또는 거짓(False)을 나타내는 자료형

```python
print(True)
print(False)
# 실행 결과
# True
# False
```

### 비교 연산자

숫자나 문자의 값을 비교하는 연산자

```python
print(3 < 5)    # 주어진 진술이 참이면 True
print(7 == 5)   # 주어진 진술이 거짓이면 False
print(2 >= 10)
print(5 != 10)
# 실행 결과
# True
# False
# False
# True
```

### 비교 연산자 종류

| 비교 연산자 | 의미 | 예시 | 결과 |
|---|---|---|---|
| `==` | 같다 | `3 == 3` / `3 == 5` | True / False |
| `!=` | 다르다 | `3 != 3` / `3 != 5` | False / True |
| `>` | 왼쪽이 더 크다 | `3 > 5` | False |
| `<` | 오른쪽이 더 크다 | `3 < 5` | True |
| `>=` | 왼쪽이 크거나 같다 | `3 >= 5` | False |
| `<=` | 오른쪽이 크거나 같다 | `3 <= 5` | True |

### 논리 자료형의 연산

- 논리 자료형은 True와 False만 존재
- 논리 자료형을 이용한 새로운 연산을 이용하면 복잡한 논리식을 단순화 할 수 있음
  - (예: "숫자는 5보다 큰가요?" + "숫자는 9보다 작은가요?" → "숫자는 5보다 크고 9보다 작은가요?"로 결합)

### AND : 모두 True 인가요?

각 논리가 모두 True여야 True

```python
print(3==3 and 4<=5 and 6>2)
# 3==3 : 3과 3은 같나요? -> True
# 4<=5 : 4는 5보다 작거나 같나요? -> True
# 6>2  : 6은 2보다 큰가요? -> True
# 세 항이 모두 True이므로 True!
# 실행 결과
# True
```

### OR : True가 있나요?

논리들 중 하나라도 True가 존재하면 True

```python
print(3==4 or 4<=5 or 6<2)
# 3==4 : 3과 4는 같나요? -> False
# 4<=5 : 4는 5보다 작거나 같나요? -> True
# 6<2  : 6은 2보다 작은가요? -> False
# 4<=5 가 True이므로 True!
# 실행 결과
# True
```

### NOT : 기존의 논리와 반대로

논리값을 뒤집음 (True ↔ False)

```python
print(not 3==4)
# 3==4 : 3과 4는 같나요? -> False
# False에 not을 붙였으므로 True!
# 실행 결과
# True
```

## 3. 만약에 내가 간다면 – 조건문

### 왜 조건문이 필요할까?

우리 삶에는 조건에 따라서 해야 할 일이 다른 경우가 많음

> (예시) 사용자 → 로그인 화면
> - "회원가입 정보"와 "로그인 정보" 같다면 → 홈페이지로 이동
> - "회원가입 정보"와 "로그인 정보" 다르다면 → 경고 문구 출력

### 조건문

어떠한 특정 조건에 따라서 실행되는 명령이 달라지는 구문

### if 문

만약, 변수 i가 1이라면, 변수 i를 출력해 주세요!

- `if` — 조건(`i == 1`) — 명령

조건이 True일 때, 명령 실행

```python
if 조건 :
    <수행할 명령>
    <수행할 명령>
    ...
```

```python
i = 1

if i == 1 :
    print(i)
# 실행 결과
# 1
```

if 문에 들어갈 명령어들은 같은 들여쓰기로 구분

```python
i = 1

if i == 1 :
    print(i)
# 실행 결과
# 1
```

## 4. 조건이 맞지 않는다면

### if 문에서 조건을 만족하지 못하면? — if-else 문

- 만약, 변수 i가 1이라면, 변수 i를 출력해 주세요! → `if` (조건 `i == 1`)
- 아니면, i+1을 출력해 주세요! → `else` (조건 `i != 1`)

조건이 True이면 if문, False이면 else문 실행

```python
if 조건 :
    <수행할 명령>
else :
    <수행할 명령>
```

```python
i = 1

if i == 1 :
    print(i)
else :
    print(i+1)
# 실행 결과
# 2
```

### if 문에서 조건을 만족하지 못했을 때 — if-elif 문

- 만약, 점수가 90점 이상이라면 "A"를 출력해 주세요! → `if` (조건 / 명령)
- 아닌 경우에 만약, 점수가 75점 이상이라면 "B"를 출력해 주세요! → `elif` (조건 / 명령)
- 그 이외의 경우, "C"를 출력해 주세요! → `else` (명령)

조건1이 True이면 if문, 조건1이 False이면서 조건2가 True이면 elif문 실행

```python
if 조건1 :
    <수행할 명령>
elif 조건2 :
    <수행할 명령>
```

```python
score = 78

if score >= 90 :
    print("A")
elif score >= 75 :
    print("B")  # B출력
else :
    print("C")
```

### 정리

```python
if 조건1 :
    수행할 명령(A)
elif 조건2 :
    수행할 명령(B)
elif 조건3 :
    수행할 명령(C)
...
else :
    수행할 명령(Z)
```

- [ 조건1 True ] → A 명령 실행
- [ 조건1 False ] and [ 조건2 True ] → B 명령 실행
- [ 조건1 False ] and [ 조건2 False ] and [ 조건3 True ] → C 명령 실행
- ...
- [ 모든 조건이 False ] → Z 명령 실행
