## 1. 출력 – print

### 출력

Python에서는 우리가 원하는 정보나 자료를 컴퓨터가 출력하게 할 수 있습니다!

> (도식) 컴퓨터 → `print` → 사용자

### print 기본 사용법

`print(출력할 내용)` 형태로 출력합니다.

```python
print("나의 꿈은 파이썬 정복!")
# 실행 결과
# 나의 꿈은 파이썬 정복!
```

### 여러 자료 출력하기

Q. 하나가 아닌, 여러 자료를 출력하고 싶은데, 어떻게 하면 좋을까?

A. `,`(콤마)를 이용해서 여러 자료를 출력할 수 있다!

```python
print(3, "Hello")
# 실행 결과
# 3 Hello!
```

### 여러 줄 출력하기

Q. print로 삼행시를 짓고 싶은데... 어떻게 하면 좋을까?

A. print를 여러 번 사용하자!

```python
print("비행기에 타신 승객 여러분")
print("행복한 여행 되십시오")
print("기내식은 바밤바")
# 실행 결과
# 비행기에 타신 승객 여러분
# 행복한 여행 되십시오
# 기내식은 바밤바
```

## 2. 기본 자료형

### 숫자형 (Number)

```python
3      # 정수(integer)
3.14   # 실수(float)
3+4j   # 복소수
```

- 숫자로 이루어진 자료형
- 정수나 실수 등을 다룰 수 있음
- 숫자끼리의 연산 가능

### 문자열 (String)

```python
'Hello!'
'3.14'   # 작은 따옴표 OK
"3.14"   # 큰 따옴표 OK
```

- 문자나 문자들을 늘어놓은 것
- 큰 따옴표(`" "`)와 작은 따옴표(`' '`)로 구분

### 리스트 (List)

```python
[]            # 빈 리스트
['a', 'b']
['a', 2]      # 다른 자료형을 함께!
```

- 여러 자료를 보관하는 자료형
- 다른 종류의 자료를 함께 담을 수 있음
- 자료 안에 순서 존재

### 주석 (Comment)

```python
# 주석 처리한 말들은
"""
컴퓨터가 실행하지
않아요!
"""
```

- 주석은 컴퓨터가 무시
- 한 줄 주석은 `#`
- 여러 줄 주석은 `""" """` 또는 `''' '''` 를 사용 (따옴표 세 번)

## 3. 변수

### 변수란

자료를 "그릇"에 담아서 보관하고 사용하면 편리, 변수(Variable)라고 함

> (도식) 그릇 안에 `"Rabbit"`, `20`, `[20, 5, 7]` 등의 자료를 담는 모습

### 변수 사용 방법

`변수 이름 = 자료`

```python
num = 10                      # 숫자
name = "Michael"              # 문자열
grade = ['A+', 'B+', 'A0']    # 리스트
```

### 변수 이름 짓는 법

변수 이름에는 숫자, 알파벳, 한글, 언더바(`_`) 등을 사용할 수 있음

```python
# Case 1: 알파벳 (권장)
python = "파이썬"
print(python)

# Case 2: 한글
파이썬 = "Python"
print(파이썬)

# Case 3: 알파벳 + 언더바(_) + 숫자
carrot_123 = 1234
print(carrot_123)
```

### 변수 이름 규칙

변수 이름을 짓는 대표적인 방법으로 스네이크 표기법과 카멜 표기법이 있음

```python
# 스네이크 표기법
prod_name = "milk"
prod_date = "20**-10-08"

# 카멜 표기법
prodName = "milk"
prodDate = "20**-10-08"
```

### 변수 이름 주의점

```python
1name = "Coke"
print(1name)

1234 = 5678
print(1234)

print = "Hello"
print(print)

prod name = "milk"
prod+date = "20**-10-08"
print(prod name, prod+date)
```

1. 변수 이름이 숫자로 시작하면 안 됨
2. 숫자로만 구성된 변수 이름 금지
3. 파이썬 문법에서 사용하는 예약어 사용 금지 (e.g. `print` 등)
4. 공백 문자와 연산자 사용 금지 (e.g. `+`, `-`, `%` 등)

## 4. 자료형의 연산

### 수학의 사칙연산

| 기호 | 의미 |
|---|---|
| `+` | 더하기 |
| `-` | 빼기 |
| `×` | 곱하기 |
| `÷` | 나누기 |

### 수학형 자료의 사칙연산

코드에서의 연산자: `+` 더하기, `-` 빼기, `*` 곱하기, `/` 나누기

```python
print(3 + 5)  # 8
print(3 - 5)  # -2
print(3 * 5)  # 15
print(3 / 5)  # 0.6
```

### 수학형 자료의 특수연산

- `//` 몫 연산자
- `%` 나머지 연산자
- `**` 제곱 연산자

```python
print(13 // 5)  # 2
print(13 % 5)   # 3
print(2 ** 4)   # 16
```

### 문자형 자료의 연산

- `+` 이어붙이기 (with 문자열)
- `*` 반복하기 (with 숫자)

```python
print("안녕" + "하세요")
# 안녕하세요

print("안녕" * 3)
# 안녕안녕안녕
```

## 5. 문자열/리스트 접근하기

### 질문

- [질문 1] "rescue"와 "secure"은 다른 문자열이다. 이유는 무엇일까요?
- [질문 2] `[1, 2, 3]`과 `[3, 2, 1]`은 다른 리스트이다. 이유는 무엇일까요?

→ 원소의 배치 순서가 다르기 때문

### 인덱스 (index)

문자열과 리스트 자료형은 여러 원소로 이루어져 있고 각각의 위치를 0부터 순서대로 매길 수 있음

```
"Ready"          [2, 4, 6, 8]
 01234            0  1  2  3

"Hi! elice!"   # 문자열에서는 공백문자도 인덱스에 포함
 0123456789
```

### 인덱싱

index를 이용해서 문자열이나 리스트의 특정 위치의 원소를 가져오는 방법

`String/List[ index_number ]`

```python
# alpha에서 인덱스가 1인 원소, 'e'를 출력
alpha = "Ready"
print(alpha[1])
```

### 슬라이싱

index를 이용해서 리스트나 문자열의 일부분을 가져오는 방법

`String/List[ a(시작 인덱스) : b(종료 인덱스) ]`

```python
# beta에서 2번째 원소 이상, 5번째 원소 미만을 가져온다
beta = [2, 4, 6, 8, 10, 12, 14]
print(beta[2:5])
# [6, 8, 10]
```
