## 1. 리스트 활용

### `list.append(d)`

자료 d를 리스트 마지막 원소 뒤에 추가. 오직 한 개의 자료만 넣을 수 있음

```python
a = []
b = ["a", "b", "c"]

a.append(10)
b.append("d")

print(a, b)
# 실행 결과
# [10] ["a", "b", "c", "d"]
```

### `list.insert(i, d)`

지정한 인덱스 위치 i에 자료 d를 추가. 오직 한 개의 자료만 넣을 수 있음

```python
c = [1, 2, 4, 5]
c.insert(2, 3)
print(c)
# 실행 결과
# [1, 2, 3, 4, 5]
```

### `list.remove(d)`

처음 나오는 자료 d를 제거

```python
my_list = [3, 1, 2, 3]
my_list.remove(3)
print(my_list)
# 실행 결과
# [1, 2, 3]
```

### `list.sort()`

리스트 내의 자료들을 순서대로 정렬. 숫자형은 오름차순, 문자열은 사전순(a부터 z까지)

```python
e = [6, 2, 4, 1]
f = ["carrot", "apple", "banana"]

e.sort()
f.sort()

print(e, f)
# 실행 결과
# [1, 2, 4, 6] ["apple", "banana", "carrot"]
```

## 2. 시퀀스 자료형

### 시퀀스 자료형이란

순서가 있는 자료형. 문자열, 리스트 등이 이에 속함

```python
'Hello!'
'3.14'   # 작은 따옴표 OK
"3.14"   # 큰 따옴표 OK

[]            # 빈 리스트
['a', 'b']
['a', 2]      # 다른 자료형을 함께!
```

### 시퀀스 자료형의 특징 (1) — 인덱싱·슬라이싱

원소 간의 순서가 존재하기 때문에 인덱싱, 슬라이싱이 가능

```python
a = "Hello"
b = ["e", "l", "i", "c", "e"]

print(a[1])
print(b[2:4])
# 실행 결과
# e
# ["i", "c"]
```

인덱싱, 슬라이싱을 할 때 음수를 넣거나 자리를 비우는 것이 가능

```python
a = "Hello"
b = ["e", "l", "i", "c", "e"]

# 뒤에서 1번째 원소
print(a[-1])

# 처음~3번째 슬라이싱
print(b[:4])

# 2번째~끝까지 슬라이싱
print(b[2:])
# 실행 결과
# o
# ["e", "l", "i", "c"]
# ["i", "c", "e"]
```

### 시퀀스 자료형의 특징 (2) — in 연산자

`in` 연산자로 시퀀스 안에 원소가 있는지 확인 가능

```python
a = "Hello"
b = ["e", "l", "i", "c", "e"]

print("e" in a)
print("E" in b)
# 실행 결과
# True
# False
```

### 시퀀스 자료형의 특징 (3) — len 연산자

`len` 연산자로 시퀀스 자료형의 길이 확인 가능

```python
a = "Hello World"
b = ["e", "l", "i", "c", "e"]

print(len(a))
print(len(b))
# 실행 결과
# 11
# 5
```

### 시퀀스 자료형의 특징 (4) — + 연산자

`+` 연산자를 이용하면 같은 종류의 시퀀스 자료를 이어 붙일 수 있음

```python
c = "Good" + "Day"
print(c)

d = ["H", "e"] + ["l", "l", "o"]
print(d)
# 실행 결과
# GoodDay
# ["H", "e", "l", "l", "o"]
```

### 시퀀스 자료형의 특징 (5) — * 연산자

`*` 연산자와 숫자를 이용하면 시퀀스 자료를 반복할 수 있음

```python
e = "Go!!" * 5
print(e)

f = [13, 17, 19] * 3
print(f)
# 실행 결과
# Go!!Go!!Go!!Go!!Go!!
# [13, 17, 19, 13, 17, 19, 13, 17, 19]
```

### 시퀀스 자료형의 활용

- 시퀀스 자료형은 순서가 있는 다양한 정보를 가지고 있어서 다양하게 활용할 수 있음
- 시퀀스 자료형의 공통적인 특징을 이용한 방법이 있음
  - e.g. 인덱스를 이용한 인덱싱과 슬라이싱
  - e.g. 구성하는 원소 내 특정 원소의 개수 구하기
- 시퀀스 자료형의 종류마다 활용법이 다를 수도 있음
  - e.g. list 자료형은 append를 이용하여 원소를 추가할 수 있지만 str 자료형은 append 사용이 불가능
  - e.g. str 자료형은 특정 문자를 기준으로 분할할 수 있지만 list 자료형은 분할 불가능
