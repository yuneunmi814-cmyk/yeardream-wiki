# Crawling (크롤링)

## 한 줄 정의

컴퓨터를 활용해 웹페이지를 **자동으로 그대로 긁어서 가져오는** 행위. 가져온 결과는 HTML 코드 형태.

## 핵심 아이디어

### 작동

```
대상 사이트 접속 → HTML 통째로 다운로드 → 저장 또는 다음 단계로 전달
```

크롤러는 결과를 보지 않고 **코드 형태 그대로** 가져온다. 의미 있는 정보로 만들려면 [[parsing]] 단계가 따로 필요.

### 강의 예시 — 다나와 상품 가격 페이지

크롤링 결과는 이런 HTML이 그대로 떨어진다:

```html
<p class="price_sect">
  <a href="https://prod.danawa.com/info/?pcode=17735156&cate=112758">
    <strong>1,899,000</strong>원
  </a>
</p>
```

→ "1,899,000원"이라는 가격을 원했지만 결과는 통째 HTML. 가격만 뽑으려면 [[parsing]] 필요.

### 포켓몬 빵 비유

```
포켓몬 빵 통째로 → 포장지와 빵은 분류 → 목표인 포켓몬 스티커만 획득
   = 크롤링         = 파싱                = 원하는 데이터
```

크롤링은 "빵 사오기"까지. 스티커 뽑기는 [[parsing]]의 일.

### 크롤링의 목적

- **시장가격 조사** (쇼핑몰 비교)
- **AI 훈련 데이터 확보**
- **가격 비교 사이트** 운영

### 비용 구조의 불균형

```
크롤링 요청 → 대상 사이트 트래픽 증가 → 유지비용 증가 / 속도 저하
```

**크롤링 트래픽 비용은 크롤링 당하는 쪽(사이트 관리자)이 부담**. 그래서 대규모·반복 크롤링은 실질적 공격에 가까워지고, 법적 분쟁의 원인이 됨.

## 합법·불법 — 회색지대

### 미국 (2022)

> "Scraping publicly accessible data is legal, according to a U.S. appeals court ruling."

- 공개적으로 접근 가능한 데이터의 스크래핑은 합법 (미국 항소법원 판결)
- 출처: techcrunch.com/2022/04/18/web-scraping-legal-court/
- 대상: 아카이비스트·연구자·저널리스트에게 의미

### 한국 — 여기어때 vs 야놀자 (정보통신망법 제48조)

- '여기어때'가 '야놀자' 제휴 숙박업소 정보 크롤링 → **무죄 확정**
- 적용 법조문: **정보통신망법 제48조** — "정당한 접근권한 없이 또는 허용된 접근권한을 넘어 정보통신망에 침입하여서는 아니된다"
- 출처: ajunews.com/view/20220515155129332
- 경계: "정당한 접근권한"의 해석이 분쟁의 핵심

### 크롤링이 사라지지 않는 이유

**데이터의 가치가 그 어느 때보다 크기 때문**. 강의 본문 표현 그대로.

## 공식 우회 루트 — 크롤링 없이 데이터 확보

- **구글 트렌드 (Google Trends)** — 전 세계 검색어 동향
- **네이버 데이터랩 (NAVER DataLab)** — 쇼핑인사이트(분야별 인기 검색어), 인기분야

→ 크롤링 합법·불법 시비 없이, 더 정제된 데이터.

## 왜 중요한가

- **프로젝트윤365 직접 영향**: 1인 컨설팅에서 "시장 동향 분석" 같은 요청은 크롤링·공식 API 사이 의사결정에 직결
- **AI 데이터셋 빌드**: Stage 3 이후 AI 프로젝트에서 데이터 확보 단계의 핵심
- **콘텐츠 후보 1순위**: "크롤링 어디까지 합법인가 — 한국·미국 판례로 보는 경계"

## 자주 헷갈리는 것

- **크롤링 vs 스크래핑**: 거의 동의어로 쓰이지만, 크롤링은 "사이트들을 돌아다니며 가져오는 행위"(링크 추적 포함), 스크래핑은 "특정 페이지에서 데이터 추출"에 가까움. 실무·강의에선 혼용.
- **크롤링 vs 파싱**: 크롤링 = 가져오기, 파싱 = 추출. 둘은 항상 짝이지만 다른 단계 ([[parsing]] 참조).
- **크롤링 방지**: `robots.txt`, IP 차단, User-Agent 필터 등으로 막을 수 있지만, 강제력은 아니라 "신사 협정"에 가까움.
