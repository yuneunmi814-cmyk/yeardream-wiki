# 조건문 (Conditional)

## 한 줄 정의

**특정 조건에 따라 실행되는 명령이 달라지는 구문**. Python에서는 `if`·`elif`·`else` + 들여쓰기로 분기.

## 핵심 아이디어

### 기본 `if`

```python
if 조건 :
    <명령>
```

```python
i = 1
if i == 1 :
    print("일")          # 조건 True일 때만
```

### `if-else`

```python
if 조건 :
    <명령>
else :
    <명령>
```

```python
score = 78
if score >= 60 :
    print("합격")
else :
    print("불합격")        # ← 출력
```

### `if-elif-else`

```python
if 조건1 :
    <명령>
elif 조건2 :
    <명령>
elif 조건3 :
    <명령>
else :
    <명령>
```

```python
score = 78

if score >= 90 :
    print("A")
elif score >= 75 :
    print("B")           # ← 출력
elif score >= 60 :
    print("C")
else :
    print("F")
```

**실행 로직**: 위에서 아래로 검사, **첫 True에서 멈춤**. 이후 elif·else 건너뜀.

### 비교 연산자 6종

| 연산자 | 의미 |
|---|---|
| `==` | 같다 |
| `!=` | 다르다 |
| `>` `<` | 크다 / 작다 |
| `>=` `<=` | 크거나 같다 / 작거나 같다 |

### 논리 연산자 3종

```python
# AND: 모두 True
age >= 18 and age < 65

# OR: 하나라도 True
day == "토요일" or day == "일요일"

# NOT: 반대로
not is_logged_in
```

### 중첩 조건문

```python
if is_member :
    if level == "VIP" :
        discount = 30
    else :
        discount = 10
else :
    discount = 0
```

→ 들여쓰기 2단으로 표현. 너무 깊으면 가독성 ↓ → 일찍 return·`elif` 활용으로 평탄화 권장.

## 왜 중요한가

- **프로그램의 첫 "지능"** — 조건문 없이는 항상 같은 흐름만
- **모든 로직의 기초** — 회원/비회원, 로그인 성공/실패, A/B 테스트 등
- **API 응답 처리** — 상태코드별 분기 (`if response.status_code == 200`)

## 자주 헷갈리는 것

### `=` vs `==` ⚠️⚠️

```python
x = 5         # 할당 (변수에 저장)
x == 5        # 비교 (같은가?)

if x = 5 :    # SyntaxError ← 할당은 조건문에 못 씀
if x == 5 :   # OK
```

→ **가장 흔한 초보 실수**.

### Python 들여쓰기

```python
if x > 0 :
    print("양수")        # 들여쓰기 (보통 스페이스 4)
print("끝")              # 들여쓰기 풀면 if 밖
```

→ Python은 들여쓰기가 **문법의 일부**. SwiftUI·JavaScript처럼 `{}` 안 씀.

### `if` 여러 개 vs `elif`

```python
# 독립 검사 (둘 다 실행 가능)
if score >= 90 : print("A")
if score >= 75 : print("B")          # 90점이면 A·B 둘 다 출력

# 배타적 (첫 True만)
if score >= 90 : print("A")
elif score >= 75 : print("B")        # 90점이면 A만 출력
```

### "True / False만 조건이 되는 건 아님"

Python은 **Truthy / Falsy** 개념:
```python
# Falsy로 취급되는 값:
0, 0.0, "", [], {}, None, False

# 그 외는 모두 Truthy
if my_list :              # 비어있으면 False, 있으면 True
    print("리스트 있음")
```

### 짧은 조건문 (한 줄)

```python
result = "양수" if x > 0 else "음수 또는 0"      # 삼항 연산자
```

→ 강의 미커버지만 실무에서 자주.

## 일상 비유: 알람 시계 메뉴

- "오전 6시?" → 일어나기 (if)
- "아니면 오전 9시?" → 운동 (elif)
- "그도 아니면?" → 자유 (else)

조건은 위에서 아래로 순서대로 확인. **한 번 맞으면 끝**.
