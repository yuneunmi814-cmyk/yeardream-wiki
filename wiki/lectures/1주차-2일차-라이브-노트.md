## 📅 2026-05-29 추가 인사이트 — 공유 문화·우분투·쉘 vs 커널

### 1. "프로그래밍 문화 = 공유 문화" ⭐⭐

→ [[Python]] "프로그래밍 본질" + [[AI-엔지니어-가이드]] "공유 문화" 섹션과 일관.
→ 공유 문화 3대 축: GitHub·StackOverflow·공식 문서/LLM.

### 2. 강사 사용 OS = 우분투 (Ubuntu) ⭐

- Ubuntu = Debian 기반 무료 배포판, AI/서버 표준
- [[리눅스-명령어-100]] = 우분투 표준 (Mac은 `apt → brew`만 다름)
- Docker로 우분투 환경 띄우면 강사와 100% 일치

### 3. 쉘 vs 커널 — 식당 비유 ⭐

```
손님 → 매니저(Shell, bash) → 주방(Kernel) → 매니저 → 손님
```

- **커널** = 주방 (CPU·메모리·디스크 실제 제어, 1개)
