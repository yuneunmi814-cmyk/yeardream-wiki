### 한 줄 요약

**리스트 메서드 4종(`append`·`insert`·`remove`·`sort`)** + **시퀀스 자료형 공통 특성 5가지** (`인덱싱·슬라이싱`·`in`·`len`·`+`·`*`). 문자열·리스트가 한 가족.

### 핵심 단어 5~7개

- [[리스트]] ⭐ — 자료를 모아두는 그릇
- **`append`** — 끝에 추가
- **`insert`** — 위치 지정 추가
- **`remove`** — 자료 삭제
- **`sort`** — 정렬
- **시퀀스 자료형** — 순서 있는 자료형 (str·list)
- **`in`·`len`** — 포함 여부·길이

### 큰 그림

```
리스트 메서드 (변경 동작):
  append(d)      → 끝에 d 추가
  insert(i, d)   → i 위치에 d 추가
  remove(d)      → 처음 나오는 d 삭제
  sort()         → 정렬 (오름차순)

시퀀스 자료형 (str·list 공통):
  [a:b] 슬라이싱
  in        → 포함 여부
  len()     → 길이
  +         → 이어붙이기
  *         → 반복

음수 인덱싱 [-1] = 마지막
```

## 📖 본문 (정독용)

> 16 슬라이드 → 2챕터.

### 챕터 1. 리스트 활용 — 메서드 4종 (슬라이드 1~8)

**키워드 (강의 본문)**:
> `list.append(d)` — 자료 d를 마지막 뒤에 추가, 한 개만
> `list.insert(i, d)` — 인덱스 i 위치에 자료 d 추가, 한 개만
> `list.remove(d)` — 처음 나오는 자료 d 제거
> `list.sort()` — 순서대로 정렬, 숫자=오름차순, 문자=사전순

✦ **강의자 풀이 추정** (LLM 보강) — [[리스트]] 페이지 ⭐

리스트는 **변경 가능 (mutable)** — 한 번 만들어도 원소 추가·제거·정렬 가능.

**`append(d)` — 끝에 추가**:

```python
a = []
b = ["a", "b", "c"]

a.append(10)         # a = [10]
b.append("d")        # b = ["a", "b", "c", "d"]

print(a, b)
```

→ **가장 자주 쓰는 메서드**. 빈 리스트로 시작해서 데이터 모을 때.

**`insert(i, d)` — 위치 지정 추가**:

```python
c = [1, 2, 4, 5]
c.insert(2, 3)       # 인덱스 2 위치에 3 끼워넣기
print(c)             # [1, 2, 3, 4, 5]
```

⚠️ **헷갈리는 점**: insert는 **"i 위치에 끼워 넣음"**. 기존 i 위치 원소는 뒤로 밀림.

**`remove(d)` — 자료 삭제**:

```python
my_list = [3, 1, 2, 3]
my_list.remove(3)    # 처음 나오는 3만 제거
print(my_list)       # [1, 2, 3]
```

⚠️ **함정**: 중복 자료 있을 때 **첫 번째만** 제거. 모두 제거하려면 반복문.

```python
# 전체 제거하려면:
while 3 in my_list:
    my_list.remove(3)
```

**`sort()` — 정렬**:

```python
e = [6, 2, 4, 1]
f = ["carrot", "apple", "banana"]

e.sort()             # [1, 2, 4, 6]      (오름차순)
f.sort()             # ['apple', 'banana', 'carrot']  (사전순)
```

**내림차순**:
```python
e.sort(reverse=True)     # [6, 4, 2, 1]
```

⚠️ **`sort()` vs `sorted()`** (강의 미언급, 보강):
- `e.sort()` — **원본 변경**, 반환값 ❌
- `sorted(e)` — **새 리스트 반환**, 원본 유지

**CRUD 매핑**:
| CRUD | 리스트 메서드 |
|---|---|
| **C**reate | `append`·`insert` |
| **R**ead | 인덱싱·슬라이싱 |
| **U**pdate | `list[i] = 새값`·`sort` |
| **D**elete | `remove`·`del list[i]`·`pop` |

→ Day 1 [[데이터베이스-기초]] CRUD와 동일 패턴.

**🎯 핵심 메시지**: 리스트는 **변경 가능**. 4종 메서드(`append·insert·remove·sort`)가 일상 도구.

### 챕터 2. 시퀀스 자료형 ⭐ (슬라이드 9~16)

**키워드 (강의 본문)**:
> 시퀀스 자료형 = 순서가 있는 자료형 (문자열·리스트)
> 특징 (1) 인덱싱·슬라이싱 (음수·자리 비우기 가능)
> 특징 (2) `in` 연산자 — 포함 여부
> 특징 (3) `len` 연산자 — 길이
> 특징 (4) `+` 연산자 — 이어붙이기 (같은 종류끼리)
> 특징 (5) `*` 연산자 — 반복

✦ **강의자 풀이 추정** (LLM 보강)

**시퀀스 자료형 = 순서 있는 자료형** ⭐. 문자열·리스트가 공통 특성 공유.

```python
'Hello!'              # 문자열 (str)
['a', 'b']            # 리스트 (list)
```

→ 둘 다 **시퀀스**라서 같은 연산 가능.

**특징 1: 인덱싱·슬라이싱 (음수·생략 가능)**:

```python
a = "Hello"
b = ["e", "l", "i", "c", "e"]

print(a[1])           # e
print(b[2:4])         # ['i', 'c']

# 음수 인덱스
print(a[-1])          # o   (마지막)

# 자리 비우기
print(b[:4])          # ['e', 'l', 'i', 'c']   (처음~3번)
print(b[2:])          # ['i', 'c', 'e']        (2번~끝)
```

⚠️ **음수 인덱스 처음 본 사람의 함정**: `[-1]`은 **마지막**, `[-2]`는 끝에서 두 번째.

**특징 2: `in` 연산자 — 포함 여부**:

```python
a = "Hello"
b = ["e", "l", "i", "c", "e"]

print("e" in a)       # True
print("E" in b)       # False  ← 대소문자 구분
```

→ 조건문에서 자주 활용. `if "error" in log_message:` 같은 패턴.

**특징 3: `len()` — 길이**:

```python
print(len("Hello World"))      # 11
print(len(["a", "b", "c"]))    # 3
print(len([]))                 # 0
```

⚠️ **공백도 1글자**. `len("Hello World") == 11` (공백 포함).

**특징 4: `+` — 이어붙이기**:

```python
c = "Good" + "Day"
print(c)              # GoodDay

d = ["H", "e"] + ["l", "l", "o"]
print(d)              # ['H', 'e', 'l', 'l', 'o']
```

⚠️ **같은 종류끼리만**. 문자열 + 리스트는 에러.

**특징 5: `*` — 반복**:

```python
e = "Go!!" * 5
print(e)              # Go!!Go!!Go!!Go!!Go!!

f = [13, 17, 19] * 3
print(f)              # [13, 17, 19, 13, 17, 19, 13, 17, 19]
```

**str vs list 차이점** (시퀀스이지만):

| 동작 | str | list |
|---|---|---|
| **인덱싱** | ✅ | ✅ |
| **슬라이싱** | ✅ | ✅ |
| **`in`·`len`** | ✅ | ✅ |
| **`+`·`*`** | ✅ | ✅ |
| **`append`** | ❌ (불변) | ✅ |
| **변경 가능** | ❌ | ✅ |
| **`split` (분할)** | ✅ | ❌ |

→ **str는 불변 (immutable)**, **list는 가변 (mutable)**. 공통 특성은 시퀀스, 차이는 가변성.

**🎯 핵심 메시지**: 시퀀스 자료형 = **str + list (+ 향후 튜플)**. 공통 5가지 특성. 단 가변성은 list만.

## 💡 일상 비유

### 비유 1: 리스트 = 빨래 바구니

- **`append`** = 빨래 옆에 추가
- **`insert(i, d)`** = 특정 위치에 끼워 넣기 (다른 빨래 밀림)
- **`remove`** = 그 빨래 빼기
- **`sort`** = 색깔·크기별 정리

### 비유 2: 시퀀스 자료형 = 줄 서 있는 사람

- **인덱싱** = 3번째 사람 누구야?
- **음수 인덱싱** = 뒤에서 첫 번째 누구야?
- **슬라이싱** = 2~5번째까지 모아줘
- **`in`** = 거기 은미 있어?
- **`len`** = 몇 명?
- **`+`** = 두 줄 합치기
- **`*`** = 같은 줄 5번 복제

### 비유 3: str vs list = 사진 vs 앨범

- **사진 (str)** = 한 번 인화하면 못 바꿈 (불변). 잘라내고 합치는 건 새 사진.
- **앨범 (list)** = 사진 추가·빼기·정리 가능 (가변).
