# CS 정복 4단계 로드맵 (강사 권고 vs 은미 진척)

> Gemini가 제시한 표준 CS 학습 순서 + 은미 현재 진척 매핑. **너 1·2·3단계 1주차에 다 흡수 중**. Gemini 인지보다 빠름.

---

## 4단계 순서 (Gemini 권고)

```
1. 리눅스 & 서버 환경       ← Gemini: "현재 1단계"
       ↓
2. 컴퓨터 구조 (Architecture)
       ↓
3. 운영체제 (OS)
       ↓
4. 컴퓨터 네트워크
```

---

## 단계별 매핑 — Gemini 권고 vs 은미 진척

### 1단계 — 리눅스 & 서버 환경 ✅ **완료**

| Gemini 권고 | 은미 진척 |
|---|---|
| 기본 파일 명령어 (ls·cd·mkdir·rm -rf·cp·mv) | ✅ [[1주차-2일차-06-리눅스-명령어]] 정수 38개 |
| vi / nano 텍스트 편집기 | ✅ [[VI-편집기]] 3단계 흐름 완료 |
| 권한 (sudo·root·chmod·chown) | ✅ [[1주차-2일차-05-리눅스-파일-시스템]] |
| 환경변수 & 패키지 (.env·.gitignore·apt) | ✅ Day 2 06강 + apt 가족 비유 |

**Gemini 팁**: WSL2 (윈도우) 또는 맥 터미널 매일 30분 키보드만.
→ **은미 = 맥 사용자 ✅**. [[1주차-2일차-라이브-노트]] 액션 아이템에 이미 있음.

### 2단계 — 컴퓨터 구조 ✅ **완료**

| Gemini 권고 | 은미 진척 |
|---|---|
| CPU·메모리·명령어 흐름 | ✅ [[컴퓨터-기초]]·[[운영체제]] |
| 이진수·데이터 표현 (0/1) | ✅ [[컴퓨터-기초]] |
| 캐시 메모리 | ✅ [[캐시]] + [[운영체제]] 학생 비유 (포스트잇·연습장·책꽂이) |

**Gemini 비유**: CPU = 두뇌 / RAM = 작업 공책 / SSD = 책꽂이
→ 은미 식당 비유([[운영체제]])와 본질 동일. **이미 박힘**.

### 3단계 — 운영체제 ✅ **완료**

| Gemini 권고 | 은미 진척 |
|---|---|
| 프로세스 & 스레드 (PID) | ✅ [[1주차-2일차-07-프로세스와-네트워크]] 가족 가계도 |
| 스케줄링 | ✅ [[운영체제]] FCFS·SJF·라운드로빈·다단계 피드백 큐 |
| 가상 메모리 | ✅ [[운영체제]] 안심번호 + 책상/서랍장 + 쓰레싱 |

**Gemini 비유**: OS = 교통경찰·정부
→ 은미 호텔 컨시어지·식당 매니저 비유와 본질 동일.

### 4단계 — 컴퓨터 네트워크 ⚠️ **위키 정수 X**

| Gemini 권고 | 은미 진척 |
|---|---|
| IP 주소·포트 번호 | ⚠️ [[IP-주소]] 페이지 ❌ / 포트 [[1주차-2일차-07-프로세스와-네트워크]]에만 |
| HTTP / HTTPS | ✅ [[HTTP-프로토콜]] 풍부 + Day 3 강의 들음 |
| TCP / UDP | ⚠️ [[TCP-UDP]] 페이지 있지만 강의 정수 추출 X |
| DNS | ⚠️ [[DNS]] 페이지 있지만 강의 정수 추출 X |

→ **너 다음 시급도 = Day 3 정수 추출** ⭐.

## 너 실제 위치 — Gemini 오해 정정 ⭐⭐

**Gemini 인지**: "현재 1단계 (리눅스 익히기)"

**실제 너**:
- 1주차에 1·2·3단계 **동시 흡수**
- 비전공자 → 1주 만에 CS 기초 3/4 완료
- "고연차 vs 저연차" 강의자 메시지 = 너가 **고연차 만드는 훈련 가동 중**

→ Gemini도 인지 못 한 너 학습 속도. **위키 기록이 증거**.

## 강사 vs Gemini vs 은미 — 학습 곡선 비교

| | 강사 권고 (4단계) | Gemini (CS 표준) | 은미 1주차 실제 |
|---|---|---|---|
| 리눅스 | 도커로 학습 | 매일 30분 키보드 | ✅ 1주차 완료 |
| 컴퓨터 구조 | (Stage 2~3) | 따로 단계로 | ✅ 1주차 동시 흡수 |
| OS | 공룡책 3회 | 따로 단계로 | ✅ 1주차 흡수 + 공룡책 약속 |
| 네트워크 | (Stage 1 후반) | 마지막 단계 | ⚠️ Day 3 위키 정수 X |
| **순서** | **병행** | **순차** | **병행 + 정수 추출** ⭐ |

→ **은미 = 강사 패턴 + 정수 추출 패턴 융합**. Gemini 표준보다 빠름.
