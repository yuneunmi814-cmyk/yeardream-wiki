### Stage 1 — OS / Linux

### Stage 1 — 네트워크 / HTTP / 보안
- [[OSI-7계층]] ⭐ — 통신을 7개 추상화 계층으로 (강의 골격, 택배 비유)
- [[TCP-IP-4계층]] — OSI 7층의 실용 버전 (응용·전송·인터넷·네트워크 액세스)
- [[TCP-UDP]] ⭐ — 신뢰성(TCP) vs 속도(UDP), 3·4-way handshake
- [[DNS]] — 도메인 → IP 변환 (인터넷 전화번호부)
- [[SSH]] — 원격 접속 보안 프로토콜 (22번 포트, public/private key)
- [[SSL-TLS]] — 통신 암호화 (HTTPS 기반)
- [[HTTP-상태코드]] — 1xx~5xx 응답 코드 분류
- [[CORS]] ⭐ — Same-Origin Policy 예외 허용 (브라우저 보안)
- [[인증과-인가]] ⭐ — Authentication vs Authorization
- [[해시]] — 단방향 암호화 (MD5·SHA, 패스워드·무결성·서명)
- [[JWT]] ⭐⭐ — JSON Web Token (모던 인증 표준, 콘텐츠 자산화 1순위)

### Stage 1 — Python 기초 (도레미 파이썬 vol 1)
- [[Python]] ⭐⭐ — 1991년 등장 고급 언어, 사용자 주력 (AI·자동화·콘텐츠)
- [[변수]] ⭐ — 자료의 "그릇", `=` 할당, 이름 규칙 4가지
- [[자료형]] — 4대 기본 (숫자·문자열·리스트·논리) + 형 변환

- [[조건문]] — if·elif·else + 비교/논리 연산자
